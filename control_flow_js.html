Control Flow (Loops and Conditionals)

Control flow in JavaScript determines the execution order of statements. It includes conditionals (if-else) and loops.

Conditional Statements

Conditional statements allow a program to make decisions.

                let age = 18;
                
                if (age >= 18) {
                    console.log("You are an adult.");
                } else {
                    console.log("You are a minor.");
                }

Loops

Loops are used to repeat actions multiple times.

For Loop

                for (let i = 0; i < 5; i++) {
                    console.log("Iteration: " + i);
                }

While Loop

                let count = 0;
                while (count < 5) {
                    console.log("Count: " + count);
                    count++;
                }

Do-While Loop

                let num = 0;
                do {
                    console.log("Number: " + num);
                    num++;
                } while (num < 5);

Understanding control flow is essential for writing efficient and logical programs in JavaScript.
