Variables and Data Types

Variables in JavaScript are used to store data values. JavaScript provides different types of variables that define the kind of data that can be stored.

Variable Declarations

  • var: The old way to declare variables, has function scope.
  • let: A modern way to declare variables, has block scope.
  • const: Used for constant values that cannot be reassigned.

Example of Variable Declaration

                var name = "John";
                let age = 25;
                const PI = 3.14;

Data Types in JavaScript

  • String: Represents text, enclosed in quotes ("hello").
  • Number: Represents numeric values (42, 3.14).
  • Boolean: Represents true or false values.
  • Array: A collection of values stored in a list.
  • Object: A collection of key-value pairs.
  • Null: Represents an empty value.
  • Undefined: A variable that has been declared but not assigned a value.

Example of Data Types

                let text = "Hello"; // String
                let number = 100; // Number
                let isTrue = true; // Boolean
                let colors = ["red", "green", "blue"]; // Array
                let person = {name: "Alice", age: 30}; // Object
                let emptyValue = null; // Null
                let notAssigned; // Undefined

Understanding variables and data types is crucial for working with JavaScript effectively.
