Variables and Data Types

In Python, variables are used to store values. A variable is created the moment you assign a value to it.

Python has several built-in data types, including:

  • int - Integer numbers (e.g., 10, -5, 1000)
  • float - Decimal numbers (e.g., 3.14, -2.5, 0.99)
  • str - Strings (e.g., "Hello, Python!", 'World')
  • bool - Boolean values (True or False)
  • list - Ordered, mutable collection (e.g., [1, 2, 3])
  • tuple - Ordered, immutable collection (e.g., (1, 2, 3))
  • dict - Key-value pairs (e.g., {"name": "Alice", "age": 25})

Variables do not require explicit declaration; Python automatically determines the type based on the assigned value.
