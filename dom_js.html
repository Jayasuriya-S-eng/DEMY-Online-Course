DOM Manipulation

The Document Object Model (DOM) allows JavaScript to interact with and modify HTML and CSS dynamically.

Selecting Elements

We can select elements using different methods.

                // Selecting by ID
                let element = document.getElementById("myElement");
                
                // Selecting by Class
                let elements = document.getElementsByClassName("myClass");
                
                // Query Selector
                let firstElement = document.querySelector(".myClass");

Modifying Elements

We can change text, styles, and attributes.

                let element = document.getElementById("myElement");
                element.innerText = "New Text";
                element.style.color = "blue";

Event Listeners

Events allow interaction with elements.

                let btn = document.getElementById("myButton");
                btn.addEventListener("click", function() {
                    alert("Button Clicked!");
                });

Mastering DOM manipulation is crucial for dynamic web applications.
