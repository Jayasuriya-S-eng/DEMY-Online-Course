Functions and Modules

Functions in Python allow code reuse and modularity by encapsulating reusable code blocks.

Defining a Function

A function is defined using the def keyword.

def greet(name): return "Hello, " + name print(greet("Alice"))

Modules

Modules allow you to organize Python code into separate files and reuse them.

# mymodule.py def add(a, b): return a + b # main.py import mymodule print(mymodule.add(5, 3))

Using functions and modules improves code structure, readability, and maintainability.
