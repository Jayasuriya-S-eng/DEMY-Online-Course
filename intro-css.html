HTML Elements & Attributes

1. HTML Elements

Elements are the building blocks of an HTML document. They consist of an opening tag, content, and a closing tag.

<p>This is a paragraph.</p>

2. HTML Attributes

Attributes provide additional information about an element. They are always inside the opening tag.

<a href="https://www.example.com">Click here</a>

Common Attributes

  • href: Specifies the URL for a link (<a href="URL">)
  • src: Defines the source for images (<img src="image.jpg">)
  • alt: Provides alternative text for images
  • id: Uniquely identifies an element
  • class: Groups multiple elements for styling

3. Forms and Input Elements

Forms are used to collect user input. They consist of various input elements like text fields, checkboxes, radio buttons, and submit buttons.

<form action="submit-form.html" method="POST">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <br>
  <label for="email">Email:</label>
  <input type="email" id="email" name="email">
  <br>
  <input type="submit" value="Submit">
</form>

Common Input Types

  • text: A single-line text field
  • email: An input field for email addresses
  • password: A field that hides the input
  • radio: A radio button for selecting one option
  • checkbox: A checkbox for selecting multiple options
  • submit: A button to submit the form
