Forms and Input Elements

What are Forms?

Forms in HTML are used to collect user input. They contain various input elements like text fields, checkboxes, radio buttons, and submit buttons.

Common Input Elements

  • Text Input: Used to enter text.
  • Password Input: Hides the text entered.
  • Email Input: Validates email addresses.
  • Radio Buttons: Select one option from multiple choices.
  • Checkboxes: Select multiple options.
  • Dropdown (Select): Choose one option from a list.
  • Textarea: Enter multiple lines of text.
  • Submit Button: Sends form data.

Basic Form Example

<form action="#" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required>

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>

  <button type="submit">Submit</button>
</form>
