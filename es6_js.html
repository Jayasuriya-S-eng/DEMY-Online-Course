ES6+ Features

ECMAScript 6 (ES6) and later versions introduced powerful features to improve JavaScript development.

1. let and const

ES6 introduced let and const for variable declaration, replacing var.

                let name = "John";
                const age = 30;

2. Arrow Functions

Simplified function syntax with =>.

                const add = (a, b) => a + b;
                console.log(add(5, 3)); // Output: 8

3. Template Literals

Allows embedding expressions in strings.

                let name = "Alice";
                console.log(`Hello, ${name}!`);

4. Destructuring

Extract values from arrays or objects easily.

                const person = { name: "John", age: 30 };
                const { name, age } = person;
                console.log(name, age);

5. Spread and Rest Operators

Used for expanding or collecting elements.

                const numbers = [1, 2, 3];
                const newNumbers = [...numbers, 4, 5];
                console.log(newNumbers);

6. Promises

Handle asynchronous operations.

                const fetchData = new Promise((resolve, reject) => {
                    setTimeout(() => resolve("Data loaded"), 2000);
                });
                fetchData.then(data => console.log(data));

ES6+ introduced many more features such as classes, modules, and async/await, making JavaScript more powerful.
