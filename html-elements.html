HTML Elements & Attributes

1. HTML Elements

Elements are the building blocks of an HTML document. They consist of an opening tag, content, and a closing tag.

<p>This is a paragraph.</p>

2. HTML Attributes

Attributes provide additional information about an element. They are always inside the opening tag.

<a href="https://www.example.com">Click here</a>

Common Attributes

  • href: Specifies the URL for a link (<a href="URL">)
  • src: Defines the source for images (<img src="image.jpg">)
  • alt: Provides alternative text for images
  • id: Uniquely identifies an element
  • class: Groups multiple elements for styling
