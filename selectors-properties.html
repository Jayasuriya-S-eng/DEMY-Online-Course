Introduction to CSS

CSS (Cascading Style Sheets) is used to style and layout web pages. It allows you to change colors, fonts, spacing, and more.

Types of CSS

  • Inline CSS: Styles applied directly to an element using the style attribute.
  • Internal CSS: Styles defined within a <style> tag inside the HTML document.
  • External CSS: Styles written in a separate file and linked using the <link> tag.
<style>
body { background-color: lightblue; }
h1 { color: darkblue; }
</style>

CSS Selectors

Selectors are used to target HTML elements to apply styles.

  • Element Selector: Targets an HTML element (e.g., p { color: blue; }).
  • Class Selector: Targets elements with a specific class (.class-name { }).
  • ID Selector: Targets a single element with a specific ID (#id-name { }).
