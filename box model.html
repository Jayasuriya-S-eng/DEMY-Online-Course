Box Model & Positioning

Understanding the Box Model

The CSS box model is a fundamental concept in web development that dictates how elements are displayed and spaced. It consists of four main parts:

  • Content: The actual text or images within the element.
  • Padding: The space between the content and the border.
  • Border: The edge surrounding the padding (or content if no padding is present).
  • Margin: The space between this element and neighboring elements.

CSS Positioning

CSS provides several positioning methods to control the layout of elements:

  • Static: The default position of elements.
  • Relative: Moves an element relative to its normal position.
  • Absolute: Positions an element relative to its nearest positioned ancestor.
  • Fixed: Keeps an element in place relative to the viewport.
  • Sticky: Switches between relative and fixed based on scroll position.
